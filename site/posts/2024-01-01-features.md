# Features

- Markdown pages/posts with automatic metadata (title/date/tags/description/draft) derived from filenames/content
- Jinja2 layouts/partials with `pages`, `tags`, `data` globals and `url_for` helper
- Tailwind CSS pipeline, JS minification, optional image optimization
- Live-reload dev server with file watching
- Sitemap + RSS generation, pretty URLs, no plugins or frontmatter
