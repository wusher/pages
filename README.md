# My Site

Built with [Medusa](https://github.com/yourname/medusa-ssg).

## Development

```bash
medusa serve
```

Open http://localhost:4000

## Build

```bash
medusa build
```

Output goes to `output/`.

## Structure

```
site/           # Content and templates
assets/         # CSS, JS, images
data/           # YAML data files
output/         # Generated site (git-ignored)
```
