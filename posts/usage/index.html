Usage

medusa new mysite
cd mysite
medusa build           # production build -> output/
medusa serve --drafts  # dev server with live reload on localhost:4000

Commands & Options

  • medusa new NAME — scaffold a new project in NAME/ (fails if non-empty)
  • medusa build [--drafts] — build site into output/
  • medusa serve [--drafts] [--port PORT] [--ws-port PORT] — dev server with live reload; --port overrides medusa.yaml port and defaults ws port to port+1 unless --ws-port or ws_port is set
